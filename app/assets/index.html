bari

An advanced build management tool for .NET projects.

Learn more »

Convention

..over configuration. With bari you only have to specify what's really important for building your project, by following a few simple rules when structuring the projects.

Large scale

One of the main design goals is to support build of complex moduler products, consist of a large number of projects.

Customization

The build process is easily extensible with scripted steps; if it is not enough, bari has a modular architecture easily extensible with plugins.
