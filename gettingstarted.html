bari 

An advanced build management tool for .NET projects.

Learn more »

Intallation

To install bari, first download and extract the latest binary package. You may want to add it to the PATH.

Once it's extracted, it is recommended to upgrade it to the latest version using the selfupdate command: 

bari selfupdate

Mono on OSX

Bari has experimental mono support. To install it, download the latest mono-specific binary package and unpack it.
